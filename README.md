# Веб-сервис для тестирование какстомных функциональных модулей
> Сервис предназначен для тестироваки разного уровня сложности алгоритмов / ИИ моделей / функции

## Возможности
* На данной стадии разработки веб-сервис позволит вам зугрузить уже готовые .py файлы содеражащие определеную структуру кода, прописанных как указания во строеном редакторе кода. Также сайт позволит вам загрузить готовые тесты в json файле для проверки правильности составленного кода (тока при загрзки модуля)
* Функциональные модули, именуемые дальше как модули можно прикрипрепить за определенной ролью/ролями (на одну роль отстуствует ограничения в количестве, как и наоборот). На каждого клиена можно прикрепить не ограниченное количество ролей с модлями. 
* Клиент может использовать chat gpt внутри профиля, а также использоватб подлюченныем ему модули.
* Под каждый модуль создается уникальное окошко, которое будет зависить от указаных полей разрабочиком модуля. На данный момент поле может бать текстовое так и загрузоченое поле для файла.

## Начало работы 
1. Сконируйте проект с github на локальное устройство
```
git clone https://github.com/6oT9lpa/website-for-testing-custom-functional-modules.git
```
2. Установите зависимости прооекта с помощью pip
```
pip install -r requirements.txt
```
3. Запустите проект используя команду (Python должен быть в Path)
```
python run.py
```
4. Перейдите на сайт http://127.0.0.1:5000, а также создайте и войдите в аккаунт. После чего вы попадете в профиль, от куда можно попасть прямиком в админ панели (если отсутсвует роль админа нужно перейти по ссылке http://127.0.0.1:5000/admin-panel)
[Админ панель](https://i.imgur.com/Sc24Vuq.png)
[Админ панель](https://i.imgur.com/Uln9rje.png)
[Админ панель](https://i.imgur.com/489yYMP.png)
6. Для создания нового модуля перейдите во вкладку МОДУЛИ -> создать модуль, заполните необходимые поля. Также после создания модуля вы его можно редактировать 

#### Важно: Для упрощения использования защита админ панели, а также профиля отсуствует !!!!
#### Заметка: Для использовая Chat GPT требуется ТОКЕН, без него работать он не будет.
